# Password Generator

#### By Carson Loser

#### This application provides the user with a password with specifications to their liking.

# Prerequisites and Dependencies

* When user clicks on "Generate Password", presented with input dependent prompts for password criteria.

* When presented a prompt for password length, user must choose a value between 8 and 128. Anything under or over will not work.

* Prompts following character length should be seperate choice prompts of uppercase letters, lowercase letters, numeric values, and/or special characters.

* The inputs chosen by the user should be validated and at least one prompt should be selected when all prompts are answered. 

* Once all prompts are answered, a password should be generated that fits all selected criteria.

* The password is dsiplayed on the page or written as an alert.
